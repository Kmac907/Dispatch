Operator Guide

Getting Started

Use this path to install Dispatch, verify the local environment, and run a first endpoint script.

1. Install

irm https://raw.githubusercontent.com/Kmac907/Dispatch/main/packaging/install-from-source.ps1 | iex

Use an elevated shell for all-users installation. Use the developer checkout flow in Installation when you want to keep source files locally.

2. Verify Local Prerequisites

dispatch --help
dispatch version
dispatch doctor

`doctor` checks local prerequisites and policy visibility. It does not remediate endpoint configuration.

3. Preview A Run

dispatch run ps .\Fix.ps1 --target PC001 --transport psrp --plan --output json

Planning validates local inputs and renders the execution plan before endpoint work starts.

4. Execute

dispatch run ps .\Fix.ps1 --target PC001,PC002 --transport psrp

Use `--transport winrm` for raw WinRM shell execution or `--transport psexec` where admin-share staging is available.

5. Inspect Results

dispatch logs list
dispatch logs show latest
dispatch logs tail latest --count 50

Every run writes `Admin\events.ndjson`, `Admin\results.json`, and per-target `stdout.txt` / `stderr.txt` files under `C:\ProgramData\Dispatch\Runs\<run-id>`.
