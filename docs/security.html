Security Model

Security

No Plaintext Secret Arguments

Dispatch does not support password, SAS token, or secret values as command-line flags.

Credential References

Job files, host files, and CLI overrides name credential references. The global config defines provider metadata and store locations. Resolved passwords stay in memory and are not serialized.

Script-Owned Payloads

Scripts own Blob, HTTPS, SMB, Azure Files, and installer-media downloads. Dispatch does not stage package payloads in v1.

Endpoint Policy

Dispatch does not remediate endpoint services, firewall rules, delegation, WinRM listeners, TrustedHosts, admin shares, or EDR policy.

Validation Hosts

Real lab host names are environment-specific and must stay in ignored local files such as `workflow/build/test-hosts.yml`.
