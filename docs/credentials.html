Credential References

Credentials

Dispatch resolves endpoint credentials through named references. References are safe to log; resolved secrets are never serialized.

Supported Providers

ProviderUse
`prompt`Prompt securely at runtime.
`pscredential`PowerShell module wrapper handoff only.
`dpapi_file`Local DPAPI-protected file.
`windows_credential_manager`Windows Credential Manager target.
`azure_keyvault`Azure Key Vault secret retrieval for endpoint credential password.

Commands

dispatch creds list
dispatch creds add prod-admin
dispatch creds test prod-admin
dispatch creds remove prod-admin

Runtime Example

dispatch run ps .\Fix.ps1 --target PC001 --transport psrp --credential prod-admin
