Clone And Build

git clone https://github.com/Kmac907/Dispatch.git
cd Dispatch
dotnet build .\Dispatch.sln
dotnet test .\Dispatch.sln

Run From Source

dotnet run --project .\src\Dispatch.Cli\Dispatch.Cli.csproj -- --help
dotnet run --project .\src\Dispatch.Cli\Dispatch.Cli.csproj -- doctor

Project Structure

src/       product code
tests/     automated tests
docs/      public docs and GitHub Pages site
module/    PowerShell module package surface
packaging/ source install and package scripts
workflow/  ignored local planning and validation state

Documentation Site

The GitHub Pages site is served from docs/. Public site pages should be operator-facing; internal roadmap and tracker files remain in the repository for project execution.
