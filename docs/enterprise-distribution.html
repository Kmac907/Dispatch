Enterprise Distribution

Enterprise distribution is not part of current v1 scope. This page exists because the documentation inventory reserves it for later distribution work.

Later Channels

Potential post-v1 channels:

  • Azure Artifacts PowerShell repository.
  • Private NuGet-backed PowerShell feed.
  • MSI installer.
  • WinGet/private package feed.
  • Optional binary PowerShell cmdlets.

Current V1 Boundary

V1 focuses on GitHub source install, local module installation, and optional ZIP packaging. Enterprise package feeds should be documented when they become implemented behavior.
