Outputs And Logs

Automation should read result files, not parse rich terminal output.

Default Layout

C:\ProgramData\Dispatch\Runs\<run-id>\
  Admin\
    events.ndjson
    results.json
  Targets\
    <target>\
      stdout.txt
      stderr.txt
      logs\
      artifacts\

Commands

dispatch logs list
dispatch logs show latest
dispatch logs tail latest --count 50
dispatch logs export latest --dest .\exports
dispatch logs retry latest

Automation Contract

Admin\results.json is the final reduced result. Admin\events.ndjson is the canonical event stream for replay, live rendering, and ingestion. Per-target stdout.txt and stderr.txt are raw execution output.
