Boundary

Script-Owned Payloads

Dispatch v1 prepares and runs the selected script. It does not stage installer media, package payloads, Blob content, Azure Files content, or SMB/HTTPS payloads for the script.

Pattern

Pass ordinary non-secret inputs to the script. The script downloads or copies any package payload it needs and writes its own logs/artifacts under the Dispatch run folder.

Do Not

  • Pass SAS tokens or passwords on the command line.
  • Assume Dispatch downloads installer media.
  • Write secret values to stdout, stderr, logs, or artifacts.
