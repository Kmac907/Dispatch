Roadmap

The roadmap is the implementation source of truth. The current focus remains the remaining `6.4` credential-reference work before YAML jobs and the later command surfaces.

Current Product Shape

  • PSRP and raw WinRM are current transport paths.
  • Run logs use `Admin\events.ndjson` and `Admin\results.json`.
  • Credential references are resolved through the global config model.
  • PowerShell module and packaging are planned v1 surfaces.
