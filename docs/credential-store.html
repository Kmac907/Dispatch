Credential Catalog

Credential Store Plan

The global Dispatch config is the credential reference catalog. Job files, host files, and CLI switches select a reference; `config.yml` defines what that reference means.

Global Config

dispatch:
  default_transport: psrp
  default_credential_provider: prompt

credentials:
  prod-admin:
    provider: prompt
    username: CONTOSO\prod.admin

  helpdesk-local:
    provider: dpapi_file
    username: .\helpdesk-admin
    path: C:\ProgramData\Dispatch\Credentials\helpdesk-local.cred

  kv-prod-admin:
    provider: azure_keyvault
    username: CONTOSO\prod.admin
    vault_uri: https://contoso-dispatch-kv.vault.azure.net/
    secret_name: prod-admin-password
    auth: default_azure_credential

Rule

No passwords in YAML. Ever. Reference data and provider locations are allowed; resolved credentials are never serialized.
