Transports

Dispatch makes transport choice explicit. Each transport maps into the same run/result model.

TransportUse WhenPrimary Requirements
psrpYou want PowerShell remoting semantics and stream capture.PowerShell remoting enabled, WSMan reachable, session config permits caller.
winrmYou want raw WS-Management shell/command execution without SMB staging.WinRM listener reachable and shell access permitted.
psexecYou need PsExec behavior in environments where admin shares are available.Admin rights, SMB/admin shares, PsExec allowed by policy.

Examples

dispatch run ps .\Fix.ps1 --target PC001 --transport psrp
dispatch run cmd whoami --target PC001 --transport winrm
dispatch run ps .\Fix.ps1 --target PC001 --transport psexec

Endpoint Policy

Dispatch reports endpoint readiness and failures. It does not silently enable remoting, firewall rules, delegation, trusted hosts, or admin shares.
