Installation

Dispatch v1 installs from source and packages the resulting executable with the PowerShell module wrapper.

Prerequisites

  • Windows 10/11 or Windows Server.
  • PowerShell 7 or Windows PowerShell 5.1.
  • .NET SDK matching global.json.
  • Git on PATH.
  • Network access to GitHub.

Source Install

irm https://raw.githubusercontent.com/Kmac907/Dispatch/main/packaging/install-from-source.ps1 | iex

The installer creates a temporary checkout, builds dispatch.exe, assembles the module, validates the installed command surface, and cleans up temporary source files.

Developer Checkout

git clone https://github.com/Kmac907/Dispatch.git
cd Dispatch
.\packaging\install-from-source.ps1 -NoCleanup

Verify

dispatch --help
dispatch version
dispatch doctor
Import-Module Dispatch
Get-Command -Module Dispatch

Installed Surface

The source install validates the dispatch command and the PowerShell module wrapper before reporting success.
