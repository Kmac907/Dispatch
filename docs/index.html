Built For Endpoint Operations

Dispatch keeps the operating model simple: choose targets, choose a transport, execute the script or command, then read a stable result.

Explicit Transports

Use PSRP, raw WinRM, or PsExec intentionally. Transport behavior is visible in the plan, logs, and per-target metadata.

Honest Run Visibility

The live dashboard reports real target states, elapsed phase time, aggregate counts, and measurable transfer progress.

Durable Outputs

Every run writes a canonical event stream, final summary, per-target stdout/stderr, and copied script-created artifacts.

Source Install For V1

The v1 installer builds Dispatch from the GitHub source, installs the module wrapper and bundled executable, validates the result, and cleans up the temporary checkout.
