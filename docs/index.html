Script-First Execution

Run PowerShell scripts and commands through the same job, target, result, and logging model.

Transport Visibility

Select PSRP, raw WinRM, or PsExec intentionally. Transport behavior is visible in planning, logs, and target metadata.

Durable Run Records

Each run writes Admin\events.ndjson, Admin\results.json, target stdout/stderr, and copied script-created artifacts when present.

Install From Source

The v1 install flow builds from the GitHub source, installs the module wrapper and bundled executable, validates the install, and cleans up the temporary checkout.

  • Use an elevated shell for all-users installation.
  • Run `dispatch doctor` after install to check local prerequisites.
  • Use the direct `dispatch` command as the canonical entry point.
