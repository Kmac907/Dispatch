Wrapper Surface

PowerShell Module

The PowerShell module is a thin wrapper over the bundled dispatch.exe. The direct dispatch command remains the canonical engine and command surface.

Expected Commands

Invoke-DispatchPowerShell
Invoke-DispatchCommand
Invoke-DispatchExecutable
Invoke-DispatchJob
Test-Dispatch
Get-DispatchVersion

Example

Import-Module Dispatch
Invoke-DispatchPowerShell -Script .\Fix.ps1 -Target PC001 -Transport psrp

Credential Handoff

The module may supply a PowerShell PSCredential for references configured with provider: pscredential. That provider is valid only through the wrapper because the password must not become a command-line argument.

Automation Rule

Wrapper commands should consume structured output or explicit result files. They should not parse rich terminal rendering.
